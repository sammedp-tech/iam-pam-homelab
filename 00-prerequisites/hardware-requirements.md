# Hardware Requirements

This lab is designed to run entirely on a single personal machine.

## Minimum Host Requirements

| Resource | Minimum | Recommended |
|--------|--------|-------------|
| CPU | 4 cores | 8 cores |
| Memory (RAM) | 16 GB | 24–32 GB |
| Disk Space | 80 GB free | 120+ GB free |
| Host OS | Windows 10/11 or Linux | Windows 11 |

> ⚠️ If your system has only 16 GB RAM, the lab will still work.  
> Start fewer VMs at a time during early phases.

---

## Virtual Machines Overview

| VM Name | Operating System | RAM | Disk |
|------|------------------|-----|------|
| Domain Controller | Windows Server 2022 | 4 GB | 40 GB |
| Windows Member Server | Windows Server 2022 | 4 GB | 40 GB |
| Linux Server | Ubuntu Server 22.04 LTS | 2 GB | 20 GB |

---

## Notes

- No GPU is required
- SSD storage is highly recommended for better VM performance
- Internet access is required **only** for ISO downloads

This lab intentionally keeps resource requirements realistic and accessible.
